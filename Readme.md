# simple-webcam-app

Simple test to get webcam feed in a video tag in HTML5.

Check demos-
- [webcam](https://mridulganga.github.io/simple-webcam-app/index.html)
- [screen](https://mridulganga.github.io/simple-webcam-app/screen.html)
